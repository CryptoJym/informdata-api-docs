Authentication

The InformData Connect API uses JWT (JSON Web Token) authentication. All API requests must include a valid JWT token in the Authorization header.

Authentication Flow

1

Get Token

POST credentials to /api/v1/authtoken

→
2

Receive JWT

Valid for 3 hours

→
3

Make Requests

Include token in headers

→
4

Refresh Token

Before expiration

Get Authentication Token

curl -X POST https://api.informdata.com/api/v1/authtoken \
  -H "Content-Type: application/json" \
  -d '{
    "username": "your_username",
    "password": "your_password"
  }'
const response = await fetch('https://api.informdata.com/api/v1/authtoken', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    username: 'your_username',
    password: 'your_password'
  })
});

const { token } = await response.json();
console.log('JWT Token:', token);
import requests

response = requests.post(
    'https://api.informdata.com/api/v1/authtoken',
    json={
        'username': 'your_username',
        'password': 'your_password'
    }
)

token = response.json()['token']
print(f'JWT Token: {token}')

Token Expiration

JWT tokens expire after 3 hours (10,800 seconds). Implement token refresh logic to maintain uninterrupted access.

Criminal Background Checks

Criminal background checks search court records to identify any criminal history. We offer multiple types of criminal searches to meet your screening needs.

Available Criminal Check Types

County Criminal (CRIMINAL)

Search criminal records at the county courthouse level. Most thorough and accurate for specific jurisdictions.

Configuration Parameters:

Parameter Type Values Description
felonyYears integer 1-10 Years to search for felony records
felonyScope string STRICT_SCOPE, EXTENDED_SCOPE, BROAD_SCOPE Geographic scope for felony search
misdemeanorYears integer 1-10 Years to search for misdemeanor records
misdemeanorScope string STRICT_SCOPE, EXTENDED_SCOPE, BROAD_SCOPE Geographic scope for misdemeanor search
trafficLevel string NO_TRAFFIC, TRAFFIC_MISDEMEANORS_ONLY, ALL_TRAFFIC Include traffic violations

Example Request:

{
  "order": {
    "clientOrderId": "CRIM-2024-001",
    "subject": {
      "FirstName": "John",
      "LastName": "Doe",
      "Birthdate": "1990-01-01"
    },
    "product": {
      "category": "CRIMINAL",
      "felonyYears": 7,
      "felonyScope": "STRICT_SCOPE",
      "misdemeanorYears": 7,
      "misdemeanorScope": "STRICT_SCOPE",
      "trafficYears": 7,
      "trafficScope": "STRICT_SCOPE",
      "trafficLevel": "TRAFFIC_MISDEMEANORS_ONLY"
    },
    "jurisdiction": {
      "county": "King",
      "state": "WA",
      "country": "United States"
    }
  }
}

Federal Criminal (CRIMINAL_FEDERAL)

Search federal district court records for federal crimes across all US federal jurisdictions.

Federal searches cover crimes prosecuted at the federal level, including interstate crimes, federal tax violations, and crimes on federal property.

National Criminal Database (NATCRIM)

Instant search across a multi-jurisdictional database. Good for identifying potential records that need county-level verification.

Database searches should be supplemented with county-level searches for complete due diligence.

Understanding Criminal Check Responses

CLEAR

No criminal records found

RECORD

Criminal records found - review case details

POSSIBLE_RECORD

Potential match requires additional review
